# Quadcore Smart Catalog

Smart catalog and inventory platform for Quadcore for Technology.

## Current Features

- Product catalog
- Live search
- WhatsApp button
- Prepared for employee WhatsApp links
- Prepared for multi-tenant architecture

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Supabase later

## Development

```bash
npm run dev
```

# Archive | الأرشيف

**Status:** Active policy · **Last Updated:** 2026-07-19 · **Scope:** Superseded documentation

## English

Archive superseded documents only after preserving their content, rationale, original path, replacement link, and status in the refactoring report. Files are not moved merely because they are old. For this refactor, legacy documents remain at their original paths to preserve history; the audit marks their authority. Future archival moves should use Git-aware moves and update incoming links.

## العربية

لا تُنقل الوثائق المستبدلة إلى الأرشيف إلا بعد حفظ محتواها وأسبابها ومسارها الأصلي ورابط البديل وحالتها في التقرير. لا تُنقل الوثيقة لمجرد قدمها. في هذه المهمة بقيت الوثائق القديمة في مساراتها لحفظ التاريخ وحدد التدقيق مرجعيتها. ينبغي أن تستخدم عمليات الأرشفة اللاحقة نقل Git وتحدث الروابط الواردة.

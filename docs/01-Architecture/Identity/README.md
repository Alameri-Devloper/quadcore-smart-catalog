# Identity Boundary | حدود الهوية

**Status:** Planned boundary · **Last Updated:** 2026-07-19 · **Scope:** Identity domain

## English

Identity owns users, employees, authentication, authorization, and access context. Catalog consumes authorized identity context without owning it.

## العربية

تملك الهوية المستخدمين والموظفين والمصادقة والصلاحيات وسياق الوصول. يستهلك الكتالوج السياق المصرح به ولا يملكه.

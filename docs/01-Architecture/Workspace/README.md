# Workspace Boundary | حدود مساحة العمل

**Status:** Foundation · **Last Updated:** 2026-07-19 · **Scope:** Workspace domain

## English

Workspace belongs to a Company and is the operational tenant scope for configuration and repository operations. Configuration such as `autoPublishReadyProducts` belongs here, though this task implements no configuration behavior.

## العربية

تتبع مساحة العمل شركة وتمثل نطاق التشغيل للمستأجر والتهيئة وعمليات المستودعات. تنتمي إعدادات مثل `autoPublishReadyProducts` إلى هذا المجال، ولا تنفذ هذه المهمة سلوكها.

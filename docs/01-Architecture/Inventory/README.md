# Inventory Boundary | حدود المخزون

**Status:** Planned · **Last Updated:** 2026-07-19 · **Scope:** Inventory domain

## English

Inventory is independent from Catalog and will own stock, quantities, warehouses, and movements. Product may be referenced but does not contain inventory state. Multi-Warehouse support is future work.

## العربية

المخزون مجال مستقل عن الكتالوج ويملك الكميات والمستودعات والحركات. يمكنه الإشارة إلى المنتج لكن المنتج لا يحتوي حالة المخزون. دعم المستودعات المتعددة مستقبلي.

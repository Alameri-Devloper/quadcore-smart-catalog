# QSC Constitution v1.0 | دستور QSC الإصدار 1.0

**Status:** Accepted
**Last Updated:** 2026-07-19
**Scope:** Durable project principles

## English

This constitution is the authority for durable QSC product, architecture, engineering, UX, and documentation principles. ADRs record specific decisions without replacing this constitution. QSC is Responsive First: Mobile, Tablet, and Desktop are equally first-class environments, and no one environment is the default or a secondary adaptation.

## العربية

هذا الدستور هو المرجع للمبادئ الدائمة للمنتج والمعمارية والهندسة وتجربة المستخدم والتوثيق. تسجل ADR القرارات المحددة ولا تستبدل الدستور. تتبع QSC مبدأ Responsive First؛ فالجوال والجهاز اللوحي والكمبيوتر بيئات أساسية ومتساوية، ولا تعد أي منها البيئة الافتراضية أو نسخة ثانوية.

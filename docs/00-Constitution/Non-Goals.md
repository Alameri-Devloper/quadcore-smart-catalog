# Non-Goals | ما لا يشمله النطاق

**Status:** Accepted · **Last Updated:** 2026-07-19 · **Scope:** Foundation Task 0

## English

This task does not implement ERP, accounting, sales, purchasing, Multi-Warehouse Inventory, approval workflow, product versioning, marketplace sync, AI pricing, Knowledge Engine, Event Bus, or external integrations. It changes no application or database behavior.

## العربية

لا تنفذ هذه المهمة ERP أو المحاسبة أو المبيعات أو المشتريات أو المخزون متعدد المستودعات أو الموافقات أو إصدارات المنتج أو مزامنة الأسواق أو تسعير AI أو Knowledge Engine أو Event Bus أو التكاملات الخارجية، ولا تغير سلوك التطبيق أو قاعدة البيانات.

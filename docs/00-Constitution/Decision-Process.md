# Decision Process | عملية اتخاذ القرار

**Status:** Accepted · **Last Updated:** 2026-07-19 · **Scope:** Architecture decisions

## English

Identify context and constraints, discuss alternatives with affected domain owners, approve the decision, record it in an ADR, implement only the approved scope, and review implementation and documentation together. A later ADR supersedes an earlier one; it does not erase history.

## العربية

تبدأ العملية بتحديد السياق والقيود، ثم مناقشة البدائل مع أصحاب المجالات المتأثرة، واعتماد القرار وتسجيله في ADR، وتنفيذ النطاق المعتمد فقط، ومراجعة التنفيذ والتوثيق معاً. القرار اللاحق يستبدل السابق بسجل جديد ولا يمحو التاريخ.

# Values | القيم

**Status:** Accepted · **Last Updated:** 2026-07-19 · **Scope:** Shared values

## English

QSC values simplicity, reliability, extensibility, consistency, usability, maintainability, tenant isolation, and knowledge preservation. Product-data quality and employee confidence take priority over superficial speed.

## العربية

قيم QSC هي البساطة والموثوقية وقابلية التوسع والاتساق وسهولة الاستخدام والصيانة وعزل المستأجرين وحفظ المعرفة. تتقدم جودة بيانات المنتج وثقة الموظف على السرعة الشكلية.
